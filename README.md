🤖 Chatbot using Transformers (Hugging Face)

 📌 Project Overview
In this project, I created a console-based chatbot using a pre-trained transformer model from Hugging Face. The chatbot is capable of interacting with users and generating relevant responses in natural language.

🎯 Aim of the Project
The aim is to develop an intelligent chatbot that can simulate human-like conversation using NLP techniques and transformer models.

🛠️ Tools & Technologies
- Python  
- Hugging Face Transformers  
- PyTorch  
- Jupyter Notebook / Google Colab  

⚙️ Main Features
- Real-time user interaction  
- Continuous conversation support  
- Automatically generated responses  
- Stop option using "exit" or "quit"  
- Uses a pre-trained conversational model  

🔄 Process Flow
User Input → Encoding → Model Processing → Response Generation → Display Output → Repeat

 ▶️ Steps to Run the Project
1. Install required libraries:
   pip install transformers torch  

2. Open the notebook in Jupyter or Colab  

3. Run all the cells  

4. Start interacting with the chatbot  

💬 Example Interaction
Chatbot: Hello! I am your assistant.  

You: Hi  
Chatbot: Hello! How can I help you today?  

You: What is AI?  
Chatbot: Artificial Intelligence refers to the simulation of human intelligence by machines that can perform tasks such as learning, reasoning, and problem solving.

You: exit  
Chatbot:ends the conversation

📚 Model Details
The chatbot is built using the pre-trained model: microsoft/DialoGPT-medium

 ✅ Final Outcome
The chatbot successfully generates responses and maintains a basic conversation flow, showing how transformer models can be applied in real-world NLP tasks.
